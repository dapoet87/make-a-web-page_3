Tags and Elements

A tag is a hidden keyword that tells the browser how to format content. An element consists of an opening tag, content, and a closing tag.

Some examples are the bold tag, b, and the emphasis tag, em. There are other types of tags that don't need to be closed. The br tag adds a line break. The img tag is used to embed images, like I added above. The a tag is used to add links. Inline vs Block Some tags like b, em, and even br are inline tags, which means they are normally displayed without line breaks. Other tags like p and div, are block tags. That means they are their own elements and can have a specified height and width.
